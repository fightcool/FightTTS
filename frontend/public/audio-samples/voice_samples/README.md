# 音色样本目录

## 文件命名规则

文件名应该遵循以下格式：`{subcategory}_{number}.wav`

### 支持的子分类

- `male` - 男性声音
- `female` - 女性声音
- `child` - 儿童声音
- `old` - 老年声音
- `neutral` - 中性声音

## 示例文件名

```
male_01.wav      # 男性声音样本1
female_01.wav    # 女性声音样本1
child_01.wav     # 儿童声音样本1
neutral_01.wav   # 中性声音样本1
male_adult.wav   # 男性成年声音
female_young.wav # 女性年轻声音
```

## 音频要求

- 格式：WAV（推荐）、MP3、M4A、FLAC
- 采样率：44.1kHz 或更高
- 比特深度：16-bit 或更高
- 时长：5-30秒
- 质量：清晰、无背景噪音、无回声

## 使用说明

1. 将音频文件放置在此目录中
2. 确保文件名符合命名规则
3. 系统将自动扫描并分类这些文件
4. 文件名中的子分类将用于自动分类

- 避免使用有版权保护的音频内容
